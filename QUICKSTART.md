## ⚡ Super Quick Setup (5 minutes)

### 1️⃣ Test Locally (Right Now!)
1. Open `lovechat.html` in your web browser
2. Enter PIN: **143** or **1432**
3. Click "Click to Enter"
4. Start chatting! 💕

**That's it! It works offline.**

---

### 2️⃣ Deploy to GitHub (Free & Easy)

#### A. Create GitHub Account
- Go to [github.com](https://github.com)
- Click "Sign up" and follow steps
- Takes 2 minutes

#### B. Create Repository
1. Click **New** (green button)
2. Name it: `anniversary-webpage`
3. Select **Public**
4. Click **Create repository**

#### C. Upload Files
1. Click **Add file** → **Upload files**
2. Drag & drop these files:
   - `lovechat.html`
   - `README.md`
3. Click **Commit changes**

#### D. Enable GitHub Pages
1. Go to **Settings**
2. Click **Pages** (left sidebar)
3. Select **Deploy from a branch**
4. Branch: **main**
5. Click **Save**

#### E. Get Your Link
Your app is now LIVE at:
```
https://YOUR_USERNAME.github.io/anniversary-webpage/lovechat.html
```

**Send this link to your partner!**

## 🎮 How to Use

### Send a Message
1. Type in the text box
2. Press **Enter** or tap **💌** button
3. Message appears instantly!

### Send a Photo
1. Tap **📷** button
2. Choose a photo
3. It sends automatically

### Send a Sticker
1. Tap **😊** button
2. Click a sticker
3. Click **Send** in preview

### Add Emojis to Message
1. Tap **😊** button
2. Tap the **Emoji** tab
3. Click any emoji to add it
4. Send your message

### Change Your Avatar
1. Click your avatar in the header
2. Select a photo
3. Avatar updates instantly for your partner

### React to Messages
1. Double-tap any message
2. ❤️ reaction appears

## 🚨 Troubleshooting

### Messages not sending?
- Check internet connection
- Refresh the page
- Clear browser cache
- Check if Firebase is working

### Photos not loading?
- Make sure photos aren't too big
- Try uploading again
- Check internet speed

### Not seeing partner's message?
- Refresh the page
- Both use same link
- Check Firebase console

### Forgot PIN?
- Edit `lovechat.html`
- Change PIN to new number
- Redeploy to GitHub
- Tell your partner new PIN
